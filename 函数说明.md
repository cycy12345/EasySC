# ============================================================================
# 单细胞快速分析流程使用指南
# ============================================================================
#
# 功能：
#   Easy_DbFinder  — 双细胞检测与过滤（可选）
#   Easy_QC        — 质控：计算 MT/RP/HB 比例，过滤低质量细胞，输出质控图
#   Easy_Normal    — 标准化：LogNormalize + FindVariableFeatures + ScaleData
#   Easy_harmony   — 降维整合：PCA + Harmony 批次校正
#   Easy_PC        — 确定最优主成分数，输出肘部图
#   RunUMAP/TSNE   — 非线性降维（Seurat 原生，二选一）
#   Easy_clustree  — 多分辨率聚类，绘制聚类树确定最优分辨率
#   Easy_CDC       — 污染检测与校正（可选，注释前后均可）
#
